# Pin Mapping

## OLED Screen

| Screen Pin | Arduino Pin | Description  |
|:----------:|:-----------:| ------------ |
| VCC        | 5V          | Power        |
| GND        | GND         | Ground       |
| CS         | D10         |              |
| RESET      | D8          |              |
| A0         | D9          |              |
| SDA        | D11         |              |
| SCK        | D13         |              |
| LED        | 3.3V        | Backlighting |

## GPS Module

| GPS Pin     | Arduino Pin | Description  |
|:-----------:|:-----------:| ------------ |
| VCC (red)   | 5V          | Power        |
| GND (black) | GND         | Ground       |
| RX (white)  | D3          |              |
| TX (green)  | D4          |              |
